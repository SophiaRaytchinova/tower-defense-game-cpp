# Tower-defense-game-cpp
This repository follows a structured, phase-based curriculum where each stage introduces real-world programming problems and the C++ concepts needed to solve them. The goal is not just to build a game-but to understand why modern software design exists.

# Curriculum overview
1. Build system & infrastructure
2. Game loop & grip-based world
3. Structs & modular design
4. Manual memory management
5. Object-oriented design
6. Polymorphism & enemy/tower types
7. STL & algorithms
8. Graph theory & pathfinding (BFS/A*)
9. Design patterns and architecture
