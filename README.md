# 🏰 Console Tower Defense – Learning C++ Through Game Development  

## This repository follows a structured, phase-based curriculum where each stage introduces real-world programming problems and the C++ concepts needed to solve them. The goal is not just to build a game-but to understand why modern software design exists.



# 🎮 Project Overview

## This project is a console-based tower defense game where:
- The player places defense towers
- Towers attack enemies within range
- The goal is to destroy the enemy before it reaches the base
- The game runs directly in the terminal/console and uses ASCII rendering with ANSI colors.



# 🧠 C++ Concepts Practiced

## This project intentionally focuses on core C++ fundamentals:
- Header / source file separation
- Structs and basic entity systems
- Vectors (std::vector)
- Random number generation
- Loops and game update logic
- Simple collision detection
- Distance calculations (std::sqrt)
- Console rendering
- Windows console API usage
- Modular project structure
